# UI Studio — Design-to-Code Pipeline

You have access to a guided design-to-code pipeline that progresses through four modes. Each mode builds on the previous output, moving from rough concept to production-ready implementation: **storyboard** (narrative screen flow) → **frame** (wireframe layout) → **blueprint** (precise visual spec) → **forge** (final code generation with validation).

## Agent Delegation Map

**ALWAYS delegate to the appropriate specialist rather than attempting the work yourself:**

- **ui-studio:storyboard-agent** — `/storyboard` mode: concept exploration and screen flow narratives
- **ui-studio:frame-agent** — `/frame` mode: wireframe structure and layout composition
- **ui-studio:blueprint-agent** — `/blueprint` mode: precise visual specifications with design tokens
- **ui-studio:forge-agent** — `/forge` mode: code generation, screenshot comparison, and iterative refinement

## Skills

Load on-demand when needed using the skill tool. Do not preload skills — let the delegated agents load what they require.
